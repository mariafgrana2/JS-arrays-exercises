Método Join

Une los elementos de un array en una cadena con un separador especificado y la devuelve.

Sintaxis

arr.join([separator])

Leer documentación
Instrucciones

Complete la función para imprimir la cadena: This is a sentence. Copie el código a continuación en su espacio de trabajo:

let arr = ["This", "is", "a", "sentence."];
function printOutString() {
// your code here
}
printOutString();

Código
let arr = ["This", "is", "a", "sentence."];
arr.join(' ');
Resultado

Método Map

Crea un nuevo array con los resultados de aplicar la función a cada elemento del array original.

Sintaxis

var nuevo_array = arr.map(function callback(currentValue, index, array) {
// Elemento devuelto de nuevo_array
}

Leer documentación
Instrucciones

Escriba una función que:

  • Tome una matriz de números.
  • Duplique el valor de cada número de la matriz.
  • Imprima la nueva matriz actualizada.

Ejemplo: Con el arreglo [1, 2, 4, 5]. El resultado debe ser [2, 4, 8, 10]

Código
const numbers = [5, 10, 15, 20, 25];
const doubles = numbers.map((num) => num * 2);
Resultado

Método Reduce

Aplica una función reductora a cada elemento de un array y devuelve un único valor.

Sintaxis

arr.reduce(callback(acumulador, valorActual[, índice[, array]])[, valorInicial])

Leer documentación
Instrucciones

Escriba un programa para calcular la suma y el producto (multiplicación) de una matriz de números. Imprima la suma y el producto.

Ejemplo: Con un arreglo [1, 2, 3, 4] La suma es 10. El producto es 24.

Código
const sumResult = arr.reduce((acc, current) => acc + current);
const productResult = arr.reduce((acc, current) => acc * current);
Resultado

Método Filter

Crea un nuevo array con los elementos que cumplen la condición definida en la función.

Sintaxis

var newArray = arr.filter(callback(currentValue[, index[, array]])[, thisArg])

Método Includes

Determina si una matriz incluye un determinado elemento, devuelve true o false.

Sintaxis

arr.includes(searchElement[, fromIndex])

Leer documentación
Instrucciones

Cree un programa que recorra las 2 matrices; si hay cursos en común, imprímalos en la consola.

let student1Courses = ['Math', 'English', 'Programming'];
let student2Courses = ['Geography', 'Spanish', 'Programming'];

Código
const findCommonCourses = student1Courses.filter((course) => student2Courses.includes(course));
Resultado
